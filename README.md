# StreamlitApp

## Project Description
This project is an interactive data visualization dashboard built using Streamlit and Plotly. 
The application allows users to explore water quality data from Biscayne Bay and interactively 
analyze environmental trends. It also integrates NASA’s Astronomy Picture of the Day API.

## Features
- Interactive filtering using sliders and dropdowns
- Dynamic Plotly visualizations
- Descriptive statistics
- API integration
- Deployed public web application

## Dataset
- Biscayne Bay Water Quality CSV dataset
- NASA APOD API

## User Instructions
1. Select a measurement type using the dropdown.
2. Adjust the time range slider to filter data.
3. Explore visualizations and statistical summaries.
4. View NASA’s Astronomy Picture of the Day.

## AGILE Development Process
This project was developed using Agile principles:
- Sprint Planning: Defined project goals and feature milestones.
- Iterative Development: Implemented core features in phases.
- Testing & Refinement: Continuously tested UI and data flow.
- Deployment: Final version deployed using Streamlit Community Cloud.
